# Change Log

All notable changes to the "IDE-TUI Bridge" extension will be documented in this file.

## [0.1.0] - 2026-03-15

### Added
- Initial release of IDE-TUI Bridge extension
- `Workspace Sync: Open Project` command to sync workspace to terminal
- Status bar button with quick access
- Ghostty terminal adapter via AppleScript automation
- Recursive layout tree support (horizontal/vertical splits)
- `.contextsync.yml` configuration file support
- VS Code settings for terminal path and inline layout override
- Idempotent tab management (reuse existing tabs by project name)
- Output channel for sync status and error messages
- Cursor IDE support
